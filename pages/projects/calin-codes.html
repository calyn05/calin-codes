Calin.Codes

Of course, I had to create a website for myself. More specifically, I created this website as a portfolio website, to showcase some of my projects and skills.

I used HTML, CSS, and Vanilla JavaScript to create this portfolio website. I also used ViteJS as a build tool, and Firebase as a back-end service.

It is built with a mobile-first approach, making it responsive on all devices. I used CSS Grid and Flexbox to create the layout, and I like to use variables and utility classes to make the code more readable, reusable, and maintainable.

This is my latest project, but it is still a work in progress with many ideas to implement in the future.
