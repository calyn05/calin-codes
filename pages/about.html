About Me

My name is Calin! and I am a

web developer

I am based in Bucharest, Romania, and I am very passionate about creating websites and web applications that are fast, responsive on all devices and easy to use.

The web development passion started approximatelly 3 years ago when I was looking for a way to change my life and the way I was living it. I was looking for a way to make a living while doing something I love, and more specificaly for a way to be able to work from anywhere, because my previous job was pretty strict and did't allow me to have much freedom.

I was looking for a way to be able to work from anywhere, and I found it in web development. But because I didn't have any experience with web development, I started learning WordPress and SEO, and I was able to create my first website that ranked with multiple keywords on Google search results.

After this, approximately one year ago I decided that I have to step up my web development game and I quited my job. I bougth a course in my country and started learning web development from scratch, and I have to say that I am very happy with my decision. I am currently on my way to become a full-stack developer, and until now I worked with the following technologies:

  • HTML
  • CSS
  • JavaScript
  • Git
  • GitHub
  • NodeJS
  • ViteJS
  • WebPack
  • Firebase
  • Heroku

Also, as you might have noticed I am pretty familiar with WordPress.

Starting with the beggining of September 2022, I am gaining experience and skills with Angular and TypeScript. This being the current focus of my studies at the company where I follow my Full-Stack Web Developer course at.

I am opened to new opportunities of any kind, so if you have any questions or you want to work with me, please feel free to contact me using the contact form on the contact page or by sending me an email at contact@calin.codes. Also, you can find me on LinkedIn and GitHub using the social media buttons.
